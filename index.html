relay-nextjs

Relay Hooks integration for Next.js apps

Get Started

Integrated with Next.js

relay-nextjs was designed with Next.js in mind from the start. Perfectly integrated with Relay Hooks and no need to change pages not using it. 100% backwards compatible.

Designed for Server Rendering

Next.js speeds your page loads up using server-side rendering but subsequent navigations require a server-round trip. relay-nextjs keeps client-side navigation snappy with all the performance of SSR.
